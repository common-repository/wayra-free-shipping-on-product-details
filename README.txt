Show icon and shipping text on products based on product price.

== Description ==

You can show icon and two lines of shipping info static or based on product price.

For example, you can set free shipping on products expensive than $ 500, then in product cheaper than $ 500 you show "Shipping on all country" and if the price is upper $ 500 you show "Free shipping!".

You can use icon, one text line or two.

It's possible to select between 4 position in product page:
* Before product price
* After product price
* Before add to cart button
* After add to cart button

== Installation ==

= From Dashboard ( WordPress admin ) =
* plugins -> Add New
* search for 'Wayra - Free shipping on product details'
* click on Install Now and then Active.

= using FTP or similar =
* Unzip "wayra-free-shipping-on-product-details.zip" file and 
* Upload "wayra-free-shipping-on-product-details" folder to the "/wp-content/plugins/" directory.
* Activate the plugin through the "Plugins" menu in WordPress.
